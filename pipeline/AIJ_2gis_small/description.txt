Описание

Нужно сделать модель, которая будет определять тип знака на изображении.


Данные
 
В аннотациях содержатся поля:
- filename - путь до изображения со знаком
- label - метка класса к знаку на изображении

Обратите внимание!
У знаков: 3_24, 3_25, 5_15_2, 5_31 и 6_2 - есть отдельные атрибуты. Эти атрибуты в поле label отделены символом - "+", например, 3_24+100. Для знака 5_15_2 атрибут - это направление стрелки, для оставшихся знаков атрибут - это цифры на знаке.

В качестве входных данных будут даны:
- train.csv - аннотации к тренировочным изображениям
- train_images/ - папка с тренировочным изображениями
- test_public_images/ - папка с публичными тестовыми изображениями
- 5_15_2_vocab.json - расшифровка атрибутов у знака 5_15_2


Метрика

Для проверки качества будет считаться f1-score с параметром усреднения микро (average=micro).
